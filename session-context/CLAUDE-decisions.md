# Architecture Decisions

## 07/04/2026 — Shell wrapper, not Claude hook
Guard must intercept BEFORE claude launches (before tokens committed). SessionStart hooks fire after.

## 07/04/2026 — Token estimation: bytes / 43
Calibrated against Claude's own count: 31MB = 727.8K tokens. Old heuristic (bytes*0.45/4) was 5x too high. Claude tokenizes parsed messages, not raw JSONL.

## 07/04/2026 — JSONL timestamp for recency, not file mtime
File mtime gets updated on every resume attempt (even aborted). Must read last message timestamp from inside the JSONL.

## 07/04/2026 — Skip files <500KB for session selection
--continue may skip -p (print mode) sessions. Guard must match this by finding the first large interactive session, not just the newest file.

## 07/04/2026 — --append-system-prompt-file + /start for fresh sessions
-p is non-interactive print mode. --append-system-prompt-file injects Gemini summary invisibly, positional "/start" triggers the session bootstrap skill.

## 07/04/2026 — Chezmoi externals for skills with GitHub repos
5 skills (codify, phase-engine, prd-taskmaster, research-before-coding, wtf) pulled from their source repos instead of tracked inline. Keeps chezmoi repo thin.

## 07/04/2026 — mcp-server-registry stays out of chezmoi
Contains API keys, is auto-generated by Claude Code. Templating it would fight the tool. Keep it ignored, secrets injected at runtime via _cl_inject_secrets().
